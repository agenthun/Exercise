#常见细节问题

    TestExternsClass 测试类的继承(顺序: 父类static属性、代码块 -> 子类static属性、代码块 -> 父类非static属性、代码块、构造函数 -> 子类非static属性、代码块、构造函数)
    TestExternsClass2 联合测试New一个类、类的继承
